# 🗳️ VoteBuddy

**VoteBuddy** is a premium, glassmorphic web application designed to guide Indian citizens through the democratic process. From registration to casting the final ballot, VoteBuddy provides a personalized, step-by-step roadmap to ensure every voice is heard.

![VoteBuddy Preview](https://via.placeholder.com/1200x600?text=VoteBuddy+Premium+Voting+Assistant)

## ✨ Key Features

- **🌐 Native Multi-Language Support**: Real-time translation for 10 major Indian languages (English, Hindi, Bengali, Telugu, Marathi, Tamil, Urdu, Gujarati, Kannada, and Odia). No external plugins required.
- **💎 Premium Glassmorphic UI**: A state-of-the-art design system featuring frosted glass effects, smooth animations, and vibrant gradients.
- **🗺️ Personalized Roadmap**: Answer 3 simple questions to get a tailored checklist for your voting journey based on your age and location.
- **📚 Knowledge Center**: A comprehensive guide to registration, required documents, and the voting process.
- **🧠 Readiness Quiz**: Test your election knowledge with an interactive, localized quiz system.
- **🤖 AI Voting Assistant**: Get instant answers to complex election queries (Coming Soon/Integrated).

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS Modules with custom design tokens
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: CSS Transitions & Keyframes
- **State Management**: React Context API (Native i18n Engine)

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Adi140108/Vote-Buddy.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `app/`: Next.js App Router pages (Home, Onboarding, Dashboard, Learn, etc.)
- `components/`: Reusable UI components (Navbar, Footer, 3D Scenes)
- `contexts/`: Global state management including the custom `LanguageContext`.
- `public/`: Static assets and images.
